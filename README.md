# Food Fider app

## Installation and Setup Instructions

1. Clone down this repository. You will need `node` and `npm` installed globally on your machine.  
```
$ git clone https://github.com/AriaWanisu/FoodFinder.git
```
2. Installation
```bash
# cd to project path and
$ npm install --force
```
3. To Start Server
```
$ npm start
```

To Visit App:
```
localhost:3000 
```
